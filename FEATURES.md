# 🛠 Feature Deep Dive: AGROKSHA AI

## 🤖 1. Virtual Scientist (AI RAG)
- **Engine**: Groq Llama-3-70B Inference.
- **Intelligence**: Retrieval-Augmented Generation (RAG) using localized soil and crop datasets.
- **Impact**: Provides hyper-accurate advice on pest control, crop selection, and nutrient management.

## 📊 2. Mandi Pulse (Market Intel)
- **Live Tracking**: Real-time integration with Agmarknet and e-NAM APIs.
- **Predictive Pricing**: Historical trend analysis for MSP 2026 planning.
- **Coverage**: 1,000+ APMC locations across India.

## 🆔 3. Digi Farm ID
- **Sovereign Identity**: A unique digital ID for every farmer.
- **Verification**: QR-code based profile sharing for government schemes and banking.
- **Privacy**: User-controlled data sharing via Supabase Auth.

## 🌤 Smart Weather & Spray Window
- **Precision**: 1-hour interval hyper-local forecasts.
- **Spray Alert**: AI-calculated "Safe-to-Spray" windows based on humidity, wind speed, and rain probability.

## 📈 4. Multi-Level Analytics
- **Crop Risk Meter**: Visual color-coded risk assessment (High/Med/Low).
- **Investment Tracker**: Simple ledger for tracking expenses and projected ROI.
