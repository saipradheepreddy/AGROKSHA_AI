# 🛡️ Security Policy: AGROKSHA AI

## 🔒 Our Commitment
User data, especially farmer location and financial information, is treated with the highest level of security. AGROKSHA AI uses **Supabase** for secure auth and database operations, and all API keys are obfuscated using **Envied**.

## 🛡️ Reporting Vulnerabilities
If you discover a security vulnerability, please do NOT open a public issue. Instead, email us directly at:
**security@saraecosystems.com**

Our security team will respond within 48 hours and work with you to resolve the issue.

## 🚫 Safe Practices
- Never commit `.env` files.
- Never hardcode API keys in the codebase.
- Always use `build_runner` to generate secure environment classes.
