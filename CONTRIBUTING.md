# 🤝 Contributing to AGROKSHA AI

We love contributions! AGROKSHA AI is part of the **SARA Ecosystems** and follows high standards of code quality and community behavior.

## 🛠 How to Contribute
1. **Fork** the repository.
2. **Clone** your fork.
3. **Create a branch** for your feature: `git checkout -b feature/amazing-feature`.
4. **Commit** your changes using professional naming: `git commit -m "feat: add soil health analytics"`.
5. **Push** to the branch: `git push origin feature/amazing-feature`.
6. **Open a Pull Request**.

## 📏 Code Standards
- Use **Clean Architecture** principles.
- Maintain **Null Safety** across all models.
- Document all new services and functions.
- Ensure all UI components follow the **Agroksha Design System**.
